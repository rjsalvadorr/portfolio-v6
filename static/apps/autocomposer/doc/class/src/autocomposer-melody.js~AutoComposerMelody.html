    public buildSimpleMelodies(chordProgression: string[], options: Object): MelodyUnit[] source

    For a given chord progression, generate a series of melodies that fit over the progression. The melodies are sorted by smoothness, and are limited to the smoothest 100 melodies by default.

    Params:

    NameTypeAttributeDescription
    chordProgression string[]

    chord progression given by user
    options Object
    • optional

    options for melody generation
    options.raw Object
    • optional

    if true, returns output as strings (default = false)
    options.limit Object
    • optional

    if false, returns all the generated melodies, not just the top 100 (default = true)
    options.filter Object
    • optional

    if false, returns melodies that are considered too "ugly" for the default process. (default = true)

    Return:

    MelodyUnit[]

    an array of MelodyUnits
